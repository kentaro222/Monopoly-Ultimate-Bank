​🎩 Monopoly Ultimate Bank (モノポリー・アルティメット銀行)
​📌 概要
​本アプリは、ボードゲーム「モノポリー」のプレイを劇的に快適にするためのデジタル・バンカー（銀行役）支援ツールです。
煩雑なお札のやり取りや、家の在庫管理、家賃の確認をすべてスマホやタブレット上で完結させることで、ゲームのテンポを上げ、より戦略的なプレイに集中できるよう設計されています。
​✨ 主な機能
​💳 キャッシュレス資金管理: 銀行とプレイヤー間、プレイヤー同士の送金を簡単に行えます。
​🏠 物件・建築管理: 誰がどの土地を持っているか、家やホテルがいくつ建っているか（銀行の在庫数も）を自動計算します。
​📖 権利書ポップアップ: 物件名をタップするだけで、更地からホテルまでの家賃や建築費をサクッと確認できます。
​🎲 プレイ補助ツール: サイコロ機能、チャンス・共同基金カードのドロー機能を搭載しています。
​💾 オートセーブ＆画面消灯防止: プレイ中に誤ってタブを閉じてもデータは保持されます。また、長時間のプレイでも画面が暗くならない機能（Wake Lock）を備えています。
​🎮 遊び方・使い方ガイド
​1. ゲームの準備
​プレイヤーの追加: 「👤 プレイヤー追加」エリアから名前を入力して参加させます（初期資金$1500が自動で付与されます）。
​画面消灯防止の設定: 画面上部の「🔆 画面消灯を防止」にチェックを入れると、プレイ中にスマホの画面が勝手に暗くなるのを防ぎます。
​2. 基本的なターン進行
​移動: 「🎲 ツール」からサイコロを振るか、実際のサイコロを使います。
​カードを引く: ❓（チャンス）または 🧰（共同基金）マスに止まったら、該当のボタンを押して指示に従います。
​※カードのお金のやり取りや移動は、自動処理されません。表示された指示に従って手動で操作してください。
​GOマスの通過: プレイヤーカードにある緑色の + $200 (GO通過) ボタンを押すだけで、銀行から給料が支払われます。
​3. 物件の購入・家賃の支払い
​購入する時: 「🔨 競売 / 定価購入」を使用します。
​購入前に ℹ️ 詳細 ボタンを押すと、その物件の家賃や建築費を確認できます。
​定価で買う場合も、落札額に定価を入力して「確定」を押してください。銀行から資金が引かれ、権利書がプレイヤーに渡ります。
​家賃を払う時: 「💸 資金の送金」を使用します。
​支払う人 ➡ 受け取る人（または銀行）を選び、金額を入力して送金します。
​4. 経営と戦略（建築・抵当・トレード）
​家/ホテルの建築: プレイヤーの所有物件リストにある 家+ 家- ボタンを押します。資金が自動で計算され、銀行の家/ホテル在庫（全32/12個）も連動して増減します。
​抵当に入れる/解除する: 抵当 ボタンを押します。銀行から半額を受け取る処理と、解除時に「半額＋10%の利子」を支払う処理がワンタップで完了します。
​プレイヤー間取引: 「🤝 プレイヤー間取引」エリアを使います。土地の譲渡や、刑務所釈放カードの受け渡しが可能です（現金のトレードは「資金の送金」で行ってください）。
​⚠️ 注意事項・その他
​データのリセットについて: ゲームが終了して新しくやり直す場合は、画面上部の赤い 🚨 データをリセット ボタンを押してください。進行状況がすべて消去されます。
​PWA（アプリ化）について: スマートフォンやタブレットのブラウザメニューから「ホーム画面に追加」を行うことで、全画面表示のネイティブアプリのように快適に動作します。
